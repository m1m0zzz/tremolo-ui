# ハロー tremolo-ui

tremolo-uiは、オーディオアプリ向けのUIライブラリです。

以下のような特徴を持っています:

- ヘッドレスUI
- モバイル対応
- ツリーシェイキング対応
- 最小限の依存 (clsx, zustand)
