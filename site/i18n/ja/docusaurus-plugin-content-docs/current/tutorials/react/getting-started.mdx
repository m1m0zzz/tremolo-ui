# Reactではじめる

## Install @tremolo-ui/react

1. tremolo-uiをインストールします

```bash npm2yarn
npm install @tremolo-ui/react
```

2. 依存関係をインストールします

```bash npm2yarn
npm install react react-dom @emotion/react
```
