# Reactではじめる

## Install @tremolo-ui/react

1. tremolo-uiをインストールします

```bash npm2yarn
npm install @tremolo-ui/react
```

2. 依存関係をインストールします

```bash npm2yarn
npm install react react-dom
```

3. コンポーネントをインポートし、デフォルトのスタイルを適応します。

```jsx title="index.tsx"
import { Knob } from '@tremolo-ui/react'
import '@tremolo-ui/react/style/index.css'
```
