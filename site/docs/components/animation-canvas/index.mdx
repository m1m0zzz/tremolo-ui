# AnimationCanvas

A simple animatable canvas with `requestAnimationFrame()`

<GitHubLink
  path='packages/react/src/components/AnimationCanvas/index.tsx'
/>

## Import

```jsx
import { AnimationCanvas } from '@tremolo-ui/react'
```


## Features

- Adapt [devicePixelRatio](https://developer.mozilla.org/docs/Web/API/Window/devicePixelRatio).
- Reduce flicking when resized.
- Observe the parent element's dimensions.
- Simple ticker and timer
